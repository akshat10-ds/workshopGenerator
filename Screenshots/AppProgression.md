# App Progression Documentation

This document highlights the key screens and progression of the Workshop Generator application.

## 1. Create Screen
The starting point where facilitators define their project, goal, and choose a workshop recipe.
![Create Screen](file:///Users/akshat.mishra/Documents/Work/workshopGenerator/Screenshots/01_create_screen.png)

## 2. Tune Screen
After selecting a recipe, this screen allows tuning the levers (e.g., Duration, Collaboration level) and viewing the generated agenda.
![Tune Screen](file:///Users/akshat.mishra/Documents/Work/workshopGenerator/Screenshots/02_tune_screen.png)

## 3. Run Screen
The interactive facilitation mode where the workshop is conducted step-by-step.
![Run Screen](file:///Users/akshat.mishra/Documents/Work/workshopGenerator/Screenshots/03_run_screen.png)

## 4. Complete Screen
The final screen celebrating the completion of the workshop and providing the generated artifacts.
![Complete Screen](file:///Users/akshat.mishra/Documents/Work/workshopGenerator/Screenshots/04_complete_screen.png)
